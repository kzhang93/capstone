CSE 219 Object Oriented Programming & Data Structures

A$90.00

The course objective is : 1. Students can define software engineering words such as software life cycle and improvement models. 2. Students can use object-oriented design to identify classes, objects, and their relationships. 3. A student can comprehend UML diagrams for OO designs and transform them to code. 4. Students can code with ArrayLists/Vectors. 5. Students can use basic data structures including linked lists, queues, stacks, 6. Students can design and build simple GUIs. 7. Students can program utilizing inheritance, interfaces, and polymorphism. 8. Students can program appropriately using exceptions.
